# ZipZapOS!

An operating system based on a root.zip file as the root for the system, with all the files being in it.

# The shell

[The shell](shell.py) will help you navigate in the root.zip file, and make changes to it.

# Installation

1. To install and use ZipZapOS, make sure you have the lateset version of python installed, and all the [required pacakges](requirements.txt) (```pip install -r requirements.txt```)
2. Download the [shell.py](shell.py) file and run it.

# Use guide

- On startup, you will be prompted to create a username and a password, Enter in any username and password you will remember. These will be stored in the root.zip file
- You will also notice that there was a root.zip file created in the directory you ran the shell.py file. This file contains all the information, activity and files in your ZipZapOS.
> [!WARNING]
> ALL OF YOUR ZIPZAPOS IS STORED ON ROOT.ZIP, AND IF YOU WILL DELETE IT ALL YOUR DATA WILL BE GONE FOREVER!

- The root.zip file can be shared and used anywhere, just put it in the same directory as shell.py and run the shell.py file.
